# Empty My Room
> 단 한 장의 사진으로 내 가구를 3D로 떼어내고 재배치하다

**심우현 · 위영림 · 손예진**

---

## 프로젝트 소개

방 구조를 바꾸고 싶을 때, 무거운 옷장과 침대를 실제로 다 옮겨봐야만 알 수 있을까요?

**Empty My Room**은 사진 한 장만으로 방 안의 가구를 3D로 분리하고, 빈 공간에서 자유롭게 재배치할 수 있는 역방향 인테리어 시뮬레이터입니다. 기존 인테리어 앱이 "새 가구 배치"에 집중했다면, 저희는 반대로 **내 짐을 그대로 활용**하는 데 집중합니다.

---

## 핵심 기능

| 단계 | 설명 |
|------|------|
| 1️⃣ Furnished | 가구로 가득 찬 방 사진 입력 |
| 2️⃣ Empty | SAM 기술로 가구를 3D로 분리 및 배경 복원 |
| 3️⃣ Re-arranged | 빈 방에서 가구를 레고처럼 자유롭게 재배치 |
