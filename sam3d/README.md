# SAM3D — 3D 객체 변환 및 가구 재배치

Empty My Room의 **3D 처리 파이프라인**입니다.
SAM2에서 분리된 가구를 3D 객체로 변환하고, 빈 방에서 물리적으로 타당하게 재배치합니다.

---

## 담당 역할

전체 파이프라인 중 아래 두 단계를 처리합니다.

```
[입력: SAM2의 가구 마스크 + 빈 방 이미지]
      ↓
  가구 3D 변환 (2D → 3D Object)   ← 이 모듈
      ↓
  물리 기반 재배치 (Re-arrangement) ← 이 모듈
      ↓
[출력: 재배치된 방 이미지]
```

---

## 핵심 과제

**① 3D 객체 변환**
- SAM2가 추출한 2D 마스크를 3D 블록 형태로 변환
- 단일 이미지에서 깊이(depth)를 추정해 실제 가구 크기와 형태를 복원

**② 배치 타당성 (Physics-based Placement)**
- 재배치 시 가구가 공중에 뜨거나 벽을 파고들지 않도록 물리적 제어
- 바닥면 감지 및 충돌 처리로 현실적인 배치 결과 제공
