# SAM2 — 가구 분할 및 배경 복원

Empty My Room의 **2D 처리 파이프라인**입니다.
방 사진에서 가구를 정밀하게 분리하고, 가구가 있던 자리의 배경을 자연스럽게 복원합니다.

---

## 담당 역할

전체 파이프라인 중 아래 두 단계를 처리합니다.

```
[입력: 방 사진]
      ↓
  가구 영역 분할 (Segmentation)   ← SAM2
      ↓
  배경 복원 (Inpainting)          ← 이 모듈
      ↓
[출력: 빈 방 이미지]
```

---

## 핵심 과제

**① 가구 분할 (Segmentation)**
- SAM2 모델을 활용해 방 사진에서 가구 객체를 마스크로 추출
- 소파, 침대, 책상 등 복잡한 형태도 정밀하게 분리

**② 배경 복원 (Occlusion Inpainting)**
- 가구를 제거한 자리에 가려져 있던 벽지, 바닥을 자연스럽게 복원
- 복원 결과가 실제 공간과 이질감 없이 어우러지도록 처리
